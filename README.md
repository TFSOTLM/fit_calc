# 健康小助手
以“关注健康”为主题，通过计算“身体质量指数BMI”，“热量消耗”,"减肥时间"和“运动心率”五个方面帮助用户科学健身，科学减重，并提供中文和英文资源包，方便更多用户使用。

## 主要功能：
+ 身体指数计算
+ 每天热量消耗计算
+ 减肥时间计算
+ 运动心率
